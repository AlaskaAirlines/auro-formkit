<div class="note">
  <p><strong>Note:</strong> The <code>AuroElement</code> base class does not handle keyboard events. It is a superclass extending <code>LitElement</code> that provides layout, shape, and size CSS class management for components such as <code>&lt;auro-input&gt;</code> and <code>&lt;auro-menu&gt;</code>.</p>
